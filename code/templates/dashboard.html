Guidelines for testing

You will be using a very basic scale of 1-5 to score sexism. Here is some guidance on how to score.

  1. Mark the AI. When testing, score how correct the AI is when it thinks it has spotted sexism. I.e. do not directly score how sexist the text is.
  2. Be generous! The machine is not as good as human beings at spotting bias. This means if the machine is mainly correct, mark 3 or above.
  3. 1 is for wrong. Reserve a score of 1 for guesses that are entirely incorrect.
  4. Ignore context. You will come across nuanced examples where authorial intent is not sexist but the character or scenario is. Mark these as sexist.
